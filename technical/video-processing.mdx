## Toy segmentation challenges

Clean toy segmentation is fundamental to the entire pipeline. The primary challenge is separating toys from hands at contact points.

### Hand-toy contact points

**The core problem:**

When fingers overlap or grip the toy, segmentation models can:
- Include parts of the hand in the toy mask ("mask leakage")
- Exclude parts of the toy that are occluded by hands
- Create jagged edges at contact boundaries

<Warning>
SAM2 needs to cleanly separate toy from hand at contact points. Fingers overlapping the toy body are critical edge cases requiring special handling.
</Warning>

**Solution approaches:**

<Steps>

```python
import cv2
import numpy as np

def refine_toy_mask(mask, frame, erosion_kernel_size=3):
    """
    Refine mask to handle hand-toy boundaries
    """
    # Erode mask slightly at boundaries to remove hand pixels
    kernel = cv2.getStructuringElement(
        cv2.MORPH_ELLIPSE, 
        (erosion_kernel_size, erosion_kernel_size)
    )
    eroded_mask = cv2.erode(mask, kernel, iterations=1)
    
    # Optional: Use color/texture to refine boundary
    # Hands typically have different color than toys
    hand_color_mask = detect_skin_color(frame)
    refined_mask = eroded_mask & ~hand_color_mask
    
    # Smooth edges
    refined_mask = cv2.GaussianBlur(
        refined_mask.astype(float), 
        (5, 5), 
        0
    ) > 0.5
    
    return refined_mask
```

### Temporal consistency checks

Use optical flow to verify mask consistency:

```python
def enforce_temporal_consistency(masks, frames):
    """
    Ensure masks remain consistent across frames
    """
    consistent_masks = [masks[0]]
    
    for i in range(1, len(masks)):
        # Calculate optical flow from previous frame
        flow = cv2.calcOpticalFlowFarneback(
            cv2.cvtColor(frames[i-1], cv2.COLOR_BGR2GRAY),
            cv2.cvtColor(frames[i], cv2.COLOR_BGR2GRAY),
            None, 0.5, 3, 15, 3, 5, 1.2, 0
        )
        
        # Warp previous mask using flow
        h, w = masks[i].shape
        flow_map = np.column_stack([
            (np.arange(w) + flow[:,:,0].flatten()).flatten(),
            (np.arange(h) + flow[:,:,1].flatten()).flatten()
        ])
        
        warped_mask = cv2.remap(
            consistent_masks[-1],
            flow_map.reshape(h, w, 2),
            None,
            cv2.INTER_LINEAR
        )
        
        # Blend with current mask for consistency
        blended_mask = 0.7 * masks[i] + 0.3 * warped_mask
        consistent_masks.append(blended_mask > 0.5)
    
    return consistent_masks
```

</Steps>

### Mask erosion strategies

Different erosion approaches for different scenarios:

| Scenario | Erosion Strategy | Kernel Size |
|---|---|---|
| Light finger touch | Minimal erosion | 1-2 pixels |
| Full hand grip | Moderate erosion | 3-5 pixels |
| Multiple contact points | Aggressive erosion + color filtering | 5-7 pixels |
| Moving toy | Temporal smoothing priority | 2-3 pixels |

<Info>
Erosion removes pixels from mask boundaries. Too much erosion can cut into the toy itself, so careful tuning is required.
</Info>

### Compositing toy onto background

**Alpha compositing with lighting adjustments:**

```python
def composite_toy_on_background(toy_cutout, toy_mask, background, position):
    """
    Composite toy onto generated background with lighting match
    """
    x, y = position
    h, w = toy_cutout.shape[:2]
    
    # Extract background region
    bg_region = background[y:y+h, x:x+w]
    
    # Analyze background lighting at this position
    bg_brightness = np.mean(cv2.cvtColor(bg_region, cv2.COLOR_BGR2GRAY))
    toy_brightness = np.mean(cv2.cvtColor(toy_cutout, cv2.COLOR_BGR2GRAY))
    
    # Adjust toy lighting to match background
    brightness_ratio = bg_brightness / (toy_brightness + 1e-6)
    adjusted_toy = toy_cutout * brightness_ratio
    adjusted_toy = np.clip(adjusted_toy, 0, 255).astype(np.uint8)
    
    # Feather mask edges for smooth blending
    feathered_mask = cv2.GaussianBlur(
        toy_mask.astype(float), 
        (7, 7), 
        2.0
    )
    
    # Alpha composite
    alpha = feathered_mask[:, :, np.newaxis]
    composited_region = (
        adjusted_toy * alpha + 
        bg_region * (1 - alpha)
    ).astype(np.uint8)
    
    # Place back in full frame
    output = background.copy()
    output[y:y+h, x:x+w] = composited_region
    
    return output
```

**Compositing quality factors:**
- Edge feathering prevents hard cutout appearance
- Lighting adjustment for realistic integration
- Color temperature matching
- Shadow generation (future enhancement)

<Note>
Feathering the mask edges by 3-7 pixels creates a smooth transition that looks more natural than hard-edged cutouts.
</Note>

```python
def auto_detect_face_area(toy_cutout, toy_mask):
    """
    Attempt to find likely face area on toy
    """
    # Find contours to get toy shape
    contours, _ = cv2.findContours(
        toy_mask, 
        cv2.RETR_EXTERNAL, 
        cv2.CHAIN_APPROX_SIMPLE
    )
    
    if len(contours) == 0:
        return None
    
    # Get bounding box
    x, y, w, h = cv2.boundingRect(contours[0])
    
    # Heuristic: Face is typically in upper-middle third
    # (works for humanoid toys, animals)
    face_y = y + int(h * 0.25)
    face_height = int(h * 0.35)
    face_x = x + int(w * 0.3)
    face_width = int(w * 0.4)
    
    return (face_x, face_y, face_width, face_height)
```

**Hybrid approach (recommended):**
- Auto-detect as initial suggestion
- Voice agent confirms or allows adjustment
- User has final control

### Tracking eyes and mouth across frames

Once face location is established, track it through toy movement:

```python
class FaceFeatureTracker:
    def __init__(self, initial_face_region, initial_toy_mask):
        self.face_offset = self.calculate_offset_from_toy_center(
            initial_face_region, 
            initial_toy_mask
        )
    
    def calculate_offset_from_toy_center(self, face_region, toy_mask):
        """Calculate face position relative to toy center"""
        # Get toy center of mass
        M = cv2.moments(toy_mask)
        toy_cx = int(M['m10'] / (M['m00'] + 1e-6))
        toy_cy = int(M['m01'] / (M['m00'] + 1e-6))
        
        face_x, face_y, face_w, face_h = face_region
        face_cx = face_x + face_w // 2
        face_cy = face_y + face_h // 2
        
        return (face_cx - toy_cx, face_cy - toy_cy)
    
    def track_face_position(self, toy_mask):
        """Get face position in current frame based on toy position"""
        # Find current toy center
        M = cv2.moments(toy_mask)
        toy_cx = int(M['m10'] / (M['m00'] + 1e-6))
        toy_cy = int(M['m01'] / (M['m00'] + 1e-6))
        
        # Apply offset to get face position
        face_cx = toy_cx + self.face_offset[0]
        face_cy = toy_cy + self.face_offset[1]
        
        return (face_cx, face_cy)
```

**Tracking considerations:**
- Toy rotation changes relative feature positions
- Optical flow can provide additional motion hints
- Temporal smoothing prevents jittery placement

### Overlay rendering

```python
def render_eyes_and_mouth(frame, eye_images, mouth_image, face_position):
    """
    Overlay eyes and mouth onto frame
    """
    face_cx, face_cy = face_position
    
    # Position eyes
    eye_spacing = 30  # pixels between eyes
    left_eye_pos = (face_cx - eye_spacing, face_cy - 10)
    right_eye_pos = (face_cx + eye_spacing, face_cy - 10)
    
    frame = overlay_image_alpha(frame, eye_images['left'], left_eye_pos)
    frame = overlay_image_alpha(frame, eye_images['right'], right_eye_pos)
    
    # Position mouth (below eyes)
    mouth_pos = (face_cx, face_cy + 25)
    frame = overlay_image_alpha(frame, mouth_image, mouth_pos)
    
    return frame

def overlay_image_alpha(background, overlay, position):
    """Alpha blend overlay onto background"""
    x, y = position
    h, w = overlay.shape[:2]
    
    # Extract alpha channel
    if overlay.shape[2] == 4:
        alpha = overlay[:, :, 3] / 255.0
        overlay_rgb = overlay[:, :, :3]
    else:
        alpha = np.ones((h, w))
        overlay_rgb = overlay
    
    # Blend
    bg_region = background[y:y+h, x:x+w]
    blended = (
        overlay_rgb * alpha[:, :, np.newaxis] +
        bg_region * (1 - alpha[:, :, np.newaxis])
    )
    
    background[y:y+h, x:x+w] = blended.astype(np.uint8)
    return background
```

## Frame-to-frame consistency

Maintaining temporal coherence across all processing stages.

### Consistency challenges

**Sources of inconsistency:**
- Segmentation mask variation between frames
- Eye/mouth tracking jitter
- Lighting adjustments creating flicker
- Lip sync transitions that are too abrupt

### Temporal smoothing strategies

**Exponential moving average for positions:**

```python
class TemporalSmoother:
    def __init__(self, alpha=0.3):
        self.alpha = alpha  # Smoothing factor (0-1)
        self.prev_value = None
    
    def smooth(self, current_value):
        if self.prev_value is None:
            self.prev_value = current_value
            return current_value
        
        # Exponential moving average
        smoothed = (
            self.alpha * current_value + 
            (1 - self.alpha) * self.prev_value
        )
        self.prev_value = smoothed
        return smoothed

# Usage for eye tracking
eye_position_smoother = TemporalSmoother(alpha=0.3)

for frame in frames:
    raw_eye_pos = tracker.track_face_position(frame)
    smooth_eye_pos = eye_position_smoother.smooth(raw_eye_pos)
    render_eyes(frame, smooth_eye_pos)
```

**Optical flow for motion compensation:**

```python
def motion_compensated_overlay(current_frame, overlay, prev_frame, prev_overlay_pos):
    """
    Use optical flow to predict overlay position
    """
    # Calculate flow from previous to current frame
    flow = cv2.calcOpticalFlowFarneback(
        cv2.cvtColor(prev_frame, cv2.COLOR_BGR2GRAY),
        cv2.cvtColor(current_frame, cv2.COLOR_BGR2GRAY),
        None, 0.5, 3, 15, 3, 5, 1.2, 0
    )
    
    # Sample flow at previous overlay position
    prev_x, prev_y = prev_overlay_pos
    motion_vector = flow[prev_y, prev_x]
    
    # Predict new position
    predicted_pos = (
        prev_x + int(motion_vector[0]),
        prev_y + int(motion_vector[1])
    )
    
    return predicted_pos
```

**Mask temporal filtering:**

```python
def temporal_mask_filter(masks, window_size=5):
    """
    Apply median filter across time to remove mask jitter
    """
    filtered_masks = []
    
    for i in range(len(masks)):
        # Get window of masks around current frame
        start = max(0, i - window_size // 2)
        end = min(len(masks), i + window_size // 2 + 1)
        window = masks[start:end]
        
        # Median filter (take majority vote per pixel)
        stacked = np.stack(window, axis=0)
        median_mask = np.median(stacked, axis=0) > 0.5
        
        filtered_masks.append(median_mask)
    
    return filtered_masks
```

```python
def batch_process_frames(frames, batch_size=8):

    Process frames in batches for GPU efficiency
```

```python
def calculate_mask_stability(masks):
    """Measure how stable masks are frame-to-frame"""
    iou_scores = []
    for i in range(1, len(masks)):
        intersection = np.logical_and(masks[i-1], masks[i]).sum()
        union = np.logical_or(masks[i-1], masks[i]).sum()
        iou = intersection / (union + 1e-6)
        iou_scores.append(iou)
    return np.mean(iou_scores)
```
